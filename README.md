Lista 04 de Exercícios - Estrutura de Repetição Do While
Descrição

Esta lista contém 15 exercícios de lógica de programação focados no uso da estrutura de repetição do-while em JavaScript. Os exercícios exploram conceitos como:

Contagem de números e sequências numéricas

Soma, média e fatorial de números

Tabuada e divisores de um número

Verificação de números primos

Sequência de Fibonacci

Leitura de múltiplos valores até uma condição de parada

O objetivo é praticar o uso de loops do-while, garantindo que o bloco de código seja executado pelo menos uma vez, desenvolvendo raciocínio lógico, controle de repetição e manipulação de dados de entrada do usuário.
